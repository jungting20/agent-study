# Hello-Agents 챕터별 요약

> **Hello-Agents**（《从零开始构建智能体》）는 AI Native Agent를 처음부터 배우는 Datawhale 오픈소스 튜토리얼입니다.
> 16개 챕터는 크게 **5개 파트**로 구성되어 있습니다.

## 전체 학습 흐름

```
기초 이론(1~3) → 직접 만들기(4~7) → 고급 기술(8~12) → 실전 프로젝트(13~15) → 졸업(16)
```

---

## 第一部分：智能体与语言模型基础（1~3장）

| 챕터 | 제목 | 하는 일 |
|------|------|---------|
| **1장** | 初识智能体 | Agent 정의·유형·구성·동작 원리. 5분 만에 첫 Agent 구현 |
| **2장** | 智能体发展史 | ELIZA부터 LLM 기반 Agent까지 역사·패러다임 변화 |
| **3장** | 大语言模型基础 | Transformer, BPE, 임베딩, 프롬프트, LLM 한계 |

**목표:** Agent가 무엇인지, LLM이 어떻게 동작하는지 이론 기반을 다집니다.

---

## 第二部分：构建你的大语言模型智能体（4~7장）

| 챕터 | 제목 | 하는 일 |
|------|------|---------|
| **4장** | 智能体经典范式构建 | **ReAct**, **Plan-and-Solve**, **Reflection** 직접 구현 |
| **5장** | 基于低代码平台的智能体搭建 | **Coze**, **Dify**, **n8n** 등 로우코드 플랫폼으로 Agent 구축 |
| **6장** | 框架开发实践 | **AutoGen**, **AgentScope**, **LangGraph**, **CAMEL** 실습 |
| **7장** | 构建你的Agent框架 | **HelloAgents** 프레임워크를 0부터 직접 구현 |

**목표:** "쓰는 법"과 "만드는 법"을 모두 익힙니다. 7장 이후 고급 파트의 기반이 됩니다.

---

## 第三部分：高级知识扩展（8~12장）

| 챕터 | 제목 | 하는 일 |
|------|------|---------|
| **8장** | 记忆与检索 | Memory 시스템, **RAG**, 저장·검색 |
| **9장** | 上下文工程 | 장기 대화를 위한 컨텍스트 설계·관리 |
| **10장** | 智能体通信协议 | **MCP**, **A2A**, **ANP** 등 Agent 간 통신 |
| **11장** | Agentic-RL | **SFT → GRPO**로 Agent용 LLM 학습 |
| **12장** | 智能体性能评估 | 평가 지표, 벤치마크, 평가 프레임워크 |

**목표:** Agent가 "기억하고, 협업하고, 학습·평가"되도록 만드는 핵심 기술을 다룹니다.

---

## 第四部分：综合案例进阶（13~15장）

| 챕터 | 제목 | 하는 일 |
|------|------|---------|
| **13장** | 智能旅行助手 | MCP + 멀티 Agent 협업 실전 (여행 플래너) |
| **14장** | 自动化深度研究智能体 | **DeepResearch Agent** 재현·분석 |
| **15장** | 构建赛博小镇 | Agent + 게임(Godot), NPC·사회 시뮬레이션 |

**목표:** 앞에서 배운 내용을 실제 프로젝트에 통합합니다.

---

## 第五部分：毕业设计及未来展望（16장）

| 챕터 | 제목 | 하는 일 |
|------|------|---------|
| **16장** | 毕业设计 | **나만의 멀티 Agent 앱** 설계·구현 (졸업 프로젝트) |

**목표:** 학습 전체를 종합해 완성형 Agent 시스템을 만듭니다.
